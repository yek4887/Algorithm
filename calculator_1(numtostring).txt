파이썬 calculator

1. 숫자형 : 사칙연산
- division always return a floating-point
: 나눗셈 계산은 float형으로 처리됨
- 거듭제곱 : ** 연산자 사용
- 변수에 값 대입 -> 등호 사용(ex) width = 20)
** 이외에도 Decimal이나 Fraction 등의 숫자형도 지원(+복소수)

2. Text형 
- 문자, 단어, 이름, 문장 등을 모두 포함
- "!" / "rabbit" / "Paris" / "Got your back"
- single quote(' ') or double quote(" ")
***character, names, words, numeral(숫자) : single quote
***인용구를 재인용하는 법(Quote a quote)
1)인용하고 싶은 단어 앞에 \ -> 'doesn\'t'
2)double quote -> "doesn't"

*** \ 뒤에 문자가 특수 문자되지 않게: 첫 따옴표 앞에 r 첨가
->raw string 만듦

2-1 : 문자열
특징:
- + 연산자로 연결 / * 연산자로 반복
- 두 개 이상의 문자열 리터럴 작성 시 :(ex) 'literal') : 자동으로 연결됨
ex) ' I love' ' You' -> ' I love You'

>>> 변수나 표현식에는 해당되지 않음
- 따라서 변수들끼리 or 변수<>리터럴 연결 : + 연산자 사용
ex) prefix = 'py' / prefix + 'thon' = 'python'

- 문자열 : 인덱싱 가능
ex) word = 'Python' / word[0] : 'P'
**슬라이싱 : 기호를 통해 사용,
ex) word[: 2] : character from beginning to 2(0 ~ 1, 2 is excluded)
- 시작위치는 포함되지만 마지막 위치는 포함되지 않음

- 문자열 = 불변 / 즉, 인덱스로 참조한 위치에 대응시킬 수 없음(에러)
