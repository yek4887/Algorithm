****모듈, 패키지, 라이브러리

1. 모듈(module) : 파이썬 코드가 들어있는 하나의 파일
- 확장자 : .py(math.py, ...)
- 다른 파일에서 import해 사용할 수 있음

2. 패키지(package) : 여러 모듈을 포함하는 폴더(파일들의 집합)
- 관련 모듈들의 그룹화, 패키지를 통해 각 모듈에 접근 가능
- 사용하기 위해 폴더 내 __init__.py라는 파일이 있어야 함

3. 라이브러리(library): 모듈, 패키지 or 둘의 모음으로 이루어진 집합
-종류:

1) 단일 모듈 기반 라이브러리 : 단일 모듈만으로 구성됨
(ex) json.py : JSON 데이터를 처리하기 위한 기능만을 제공)

2) 다중 모듈 기반 라이브러리: 내부적으로 여러 모듈들로 구성됨
(ex) requests library : HTTP 요청을 쉽게 처리해주는 기능 포함)

3) 패키지 기반 라이브러리 : 여러 모듈과 하위 패키지로 구성됨
(ex)NumPy : //)
