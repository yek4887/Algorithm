4. break continue statements
1)break문 : break문 종료 시 진행 중이던 loop 종료
for n in range(2, 10) :
       for x in range(2, n) :
	if n % x == 0:
	       printf(f"{n} equals {x} * {n//x}")
	       break
result:     4 equals 2 * 2
	6 equals 2 * 3
	8 equals 2 * 4
	9 equals 3 * 3

2)continue문:  continues with the next iteration of the loop
for num in range(2, 10): # 2,3,4,5,..........
    if num % 2 == 0:
        print(f"Found an even number {num}")
        continue
    print(f"Found an odd number {num}")
