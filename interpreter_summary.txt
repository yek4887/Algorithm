파이썬 인터프리터

1) 대화형 모드 : 명령을 tty에서 읽을 때를 말함
(즉, 터미널에서 사용자의 입력을 실시간으로 받아들여 실행,
tty = 터미널과 연결된 표준 입력장치-> mac의 터미널, windows의
command prompt)
- tty에서 사용자가 명령 입력 시 인터프린터가 이를 받아 실행
- >>> : 파이썬 인터프린터가 사용자 입력을 기다리는 상태
ex) 사용자가 "Hello World!" 입력 -> TTY를 통해 입력된 명령을
인터프린터가 읽고 실행

2) 소스 코드 인코딩
- 파이썬 소스 파일 : UTF-8로 인코딩됨
**UTF-8 : 컴퓨터에서 문자를 저장 or 전송 시 문자 인코딩 방식 사용
특징:
1. 텍스트 데이터 ->이진 데이터로 변환
2. 유니코드 표준 기반, 모든 문자 표현 가능
3. 문자는 1~4byte 길이로 표현/ ASCII와 호환 가능
